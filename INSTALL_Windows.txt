
Installation on a non-unix PC without compilation:
   Copy dpic.exe from the Windows directory at
   https://ece.uwaterloo.ca/~aplevich/dpic
   into your folder of executable files.

   Note that, depending on the operating system, this version
   may access files using PC format for the
   file path; that is, with backslash separators.

Compiling:
   Read the INSTALL file first.
